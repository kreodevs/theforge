# docs/backstage

Documentación ejecutable para integrar The Forge con Backstage (Spotify): crear plantilla y repositorio a partir de los documentos generados por The Forge.

| Documento                                   | Uso                                                                                                                                |
| ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| **SPEC-INTEGRACION-THE-FORGE-BACKSTAGE.md** | Especificación: arquitectura, flujo, contrato Scaffold Manifest (YAML/JSON) y API Backstage Scaffolder.                            |
| **GUIA-PLANTILLA-BACKSTAGE.md**             | Guía paso a paso para crear la plantilla en Backstage (template.yaml, skeleton, catalog-info.yaml) y que el dev descargue el repo. |
| **RUNBOOK-EJECUCION-INTEGRACION.md**        | Runbook: orden de ejecución, prerrequisitos y validación (sin generar código en este paso).                                        |
