# DESIGN.md

## Design Summary

Vercel's marketing site uses a strict monochromatic palette anchored in near-black (#171717) on off-white (#fafafa/#ebebeb) surfaces. The design is defined by Geist — Vercel's own typeface — deployed at extreme negative letter-spacing for display headings (−3.84px at 64px) and tight tracking for body copy. Elevation is expressed entirely through 1px border rings (box-shadow spread) rather than drop shadows, keeping the surface language flat and precise. The hero features a bold geometric black triangle as the brand symbol, and secondary labels use all-caps spaced Geist Mono for a developer-tool aesthetic. CTAs are pill-shaped with a filled black primary and a ghost outline secondary.
