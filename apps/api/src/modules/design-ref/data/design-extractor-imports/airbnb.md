# DESIGN.md

## Design Summary

Airbnb's design system is built around a single custom variable font (Airbnb Cereal VF), a near-monochromatic neutral palette punctuated by a signature coral-red (#FF385C), and a card-grid layout system with generous rounded corners. The UI is light-themed with white surfaces, subtle multi-layer elevation shadows, and a structured search bar as the primary interaction hub. Listing cards dominate the content area with photo-first presentation, badge overlays, and compact metadata typography.
