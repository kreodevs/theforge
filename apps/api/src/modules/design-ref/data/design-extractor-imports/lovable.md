## Design Summary

Lovable's design system pairs a dramatic full-bleed aurora gradient hero (cycling through pink, magenta, blue, and orange) with a warm off-white surface palette (#eceae4, #fcfbf8). The primary typeface is Camera Plain Variable — a contemporary variable serif — used across all text roles from 14px body to 60px display. Interaction is centered on a prominent chat input card with rounded corners (12px), floating over the gradient. Navigation is minimal and transparent. The overall tone is warm, expressive, and AI-product-forward.
