# DESIGN.md

## Design Summary

Apple.com uses a tightly controlled, brand-owned typographic system (SF Pro Display + SF Pro Text) on near-white and light-gradient surfaces. The design is characterised by large, centred hero headlines, pill-shaped CTAs in Apple Blue (#0071e3), and a near-black (#1d1d1f) text palette. Elevation is conveyed through background colour shifts rather than shadows. Navigation is a translucent 44px global bar with secondary grey tones. The overall aesthetic is spacious, product-photography-led, and typographically precise.
